# Air
开源多功能机场后端， 协议支持 V2Ray(VMess), Trojan, Shadowsocks(单端口多用户)；面板支持 v2board

# 交互式安装
```
bash <(curl -Ls https://raw.githubusercontent.com/hhttco/Air/refs/heads/main/scripts/air.sh)
```

# 一键安装
节点类型 0.VMess 1.ShadowSocks 2.Trojan
```
bash <(curl -Ls https://raw.githubusercontent.com/hhttco/Air/refs/heads/main/scripts/air.sh) 面本URL 通讯密钥 节点ID(1,2,3) 节点类型(1,1,1)
```
